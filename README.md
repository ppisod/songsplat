# songsplat
a quick & easy Python library for labelling songs, music, and sound based upon perceptual dimensions (e.g. bounciness) using neural networks.
## how do I use?
```bash
git clone https://github.com/ppisod/songsplat
cd songsplat
pip install -e .
```
reload/open up another terminal
```bash
songsplat gui
```
to open up the GUI app
```bash
songsplat -h
```
to open up the help menu
